OpenLDBWS API Token Register: https://realtime.nationalrail.co.uk/OpenLDBWSRegistration/Registration Keep this in a .env file in project root dir i.e.: TRAIN_API_TOKEN=keyvalue

OpenLDBWS API Endpoint

inky_pi.configs.TRAIN_NUMBER: int = 3

Number of departing trains to fetch

OpenWeatherMap API Key Keys: https://home.openweathermap.org/api_keys Keep this in a .env file in project root dir i.e.: WEATHER_API_TOKEN=keyvalue
