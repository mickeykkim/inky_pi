inky_pi.configs module

Inky_Pi configuration settings.

inky_pi.configs.EXCLUDE_FLAGS: str = 'minutely,hourly'

Weather data exclude flags

inky_pi.configs.INKY_COLOR: str = 'yellow'

Inky color model (yellow, red, black)

inky_pi.configs.LATITUDE: float = 51.5085

Target Latitude (default in London)

inky_pi.configs.LONGITUDE: float = -0.1257

Target Longitude (default in London)

inky_pi.configs.STATION_FROM: str = 'BHO'

From (departing) station

inky_pi.configs.STATION_TO: str = 'WMW'

To (arrival) station

inky_pi.configs.TRAIN_API_TOKEN: str = 'keep-in-.env-file'

OpenLDBWS API Token Register: https://realtime.nationalrail.co.uk/OpenLDBWSRegistration/Registration Keep this in a .env file in project root dir i.e.: TRAIN_API_TOKEN=keyvalue

inky_pi.configs.TRAIN_MODEL: str = 'OPEN_LIVE'

Train model to use (OPEN_LIVE or HUXLEY2)

inky_pi.configs.TRAIN_MODEL_URL: str = 'http://lite.realtime.nationalrail.co.uk/OpenLDBWS/wsdl.aspx?ver=2017-10-01'

OpenLDBWS API Endpoint

inky_pi.configs.TRAIN_NUMBER: int = 3

Number of departing trains to fetch

inky_pi.configs.WEATHER_API_TOKEN: str = 'keep-in-.env-file'

OpenWeatherMap API Key Keys: https://home.openweathermap.org/api_keys Keep this in a .env file in project root dir i.e.: WEATHER_API_TOKEN=keyvalue

inky_pi.configs.WEATHER_MODEL: str = 'OPEN_WEATHER_MAP'

Weather model to use (OPEN_WEATHER_MAP only for now)
